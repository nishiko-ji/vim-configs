# vim-configs
userディレクトリ配下にgit cloneした場合は.vimrcに以下を記述する.
```vim
source ~/vim-configs/character.vim
source ~/vim-configs/plugin.vim
source ~/vim-configs/scheme.vim
source ~/vim-configs/statusline.vim
source ~/vim-configs/command.vim
source ~/vim-configs/indent.vim
source ~/vim-configs/search.vim
source ~/vim-configs/mode.vim
source ~/vim-configs/cursor.vim
source ~/vim-configs/clipboard.vim
source ~/vim-configs/syntastic.vim
source ~/vim-configs/completion.vim
source ~/vim-configs/ctrlp.vim
```
